# Macathon26

Root workspace for the downtown Toronto traffic + building impact project.

Primary app lives in:
- `my-react-project/`

## Tech Stack
Frontend: React, MapLibre
Backend: FastAPI

## Quick Start

From repo root:

```bash
cd my-react-project
npm install
npm run dev
```

For frontend + backend together:

```bash
cd my-react-project
npm run dev:all
```
